# WinDlx-syntax-highlight-for-Sublime-Text-3

To install:

1. place `windlx.tmLanguage` file in: `C:\Users\[...]\AppData\Roaming\Sublime Text 3\Packages\User\` folder
2. open Sublime Text 3
3. open file
4. `ctrl` + `shift` + `p`
5. type: `windlx`
6. choose: `Set Syntax: WinDlx (by MaciejLisCK)`
7. enjoy
